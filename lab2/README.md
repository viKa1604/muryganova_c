Ответы на вопросы:

1- Как объявить структуру в языке C 

Структуру в C можно объявить несколькими способами:

1. Базовое объявление с использованием `struct` и имени структуры:
struct Student {
    char name[50];
    int age;
    float average_score;
};

2. Объявление с созданием переменной:
struct {
    char name[50];
    int age;
    float average_score;
} student1;

3. Объявление с использованием typedef:
typedef struct {
    char name[50];
    int age;
    float average_score;
} Student;  

4. Объявление с тегом и typedef:
typedef struct Student {
    char name[50];
    int age;
    float average_score;
} Student;


2- Что такое вложенные структуры и зачем они нужны

Вложенные структуры — это структуры, которые содержат в качестве своих полей другие структуры.

Пример:
struct Date {
    int day;
    int month;
    int year;
};

struct Person {
    char name[50];
    struct Date birth_date;  
};

Зачем нужны:
- Логическая группировка данных — объединение связанных данных в единую сущность
- Модульность и повторное использование — структуру Date можно использовать в разных контекстах
- Упрощение кода — вместо множества отдельных полей используется одна структура
- Иерархическое представление данных — отражение реальных отношений между объектами

3- Как передать структуру в функцию

1. Передача по значению:
void printStudent(struct Student s) {
    printf("%s, %d, %.2f\n", s.name, s.age, s.avg);
}

2. Передача по указателю:
void changeAge(struct Student *s, int new_age) {
    s->age = new_age;  
}

Особенности:
- При передаче по значению создается копия всей структуры
- При передаче по указателю изменения влияют на оригинал
- Доступ к полям через указатель: ptr->field или (*ptr).field


4- Можно ли создавать массивы структур

Да, можно. Синтаксис аналогичен массивам других типов:

struct Student group[30];          
struct Student *ptr = group;       


struct Student students[] = {
    {"Иван", 20, 4.5},
    {"Мария", 19, 4.8},
    {"Петр", 21, 4.2}
};

struct Student *dynamic = malloc(10 * sizeof(struct Student));

6- Можно ли использовать enum внутри структуры

Да, можно. enum можно использовать как обычный тип данных внутри структуры:

enum Course { FIRST = 1, SECOND, THIRD, FOURTH };
enum Day { MON, TUE, WED, THU, FRI, SAT, SUN };

struct Student {
    char name[50];
    int age;
    enum Course course;   
    float average_score;
};

struct Schedule {
    char subject[50];
    enum Day lecture_day; 
    int room;
};

// Использование
struct Student s;
s.course = SECOND;



7- Можно ли сравнивать структуры оператором ==

Нет, нельзя. В языке C нельзя напрямую сравнивать структуры с помощью операторов == или !=.

struct Student s1 = {"Иван", 20, 4.5};
struct Student s2 = {"Иван", 20, 4.5};

// if (s1 == s2) { ... }  

Почему:
- В C нет встроенной поддержки сравнения сложных типов
- Структуры могут содержать поля разных типов и выравнивание (padding)

Как правильно сравнивать:

int compareStudents(struct Student *a, struct Student *b) {
    return (strcmp(a->name, b->name) == 0 && 
            a->age == b->age && 
            a->average_score == b->average_score);
}
